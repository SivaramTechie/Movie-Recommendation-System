Movie Recommendation System








Development of the Movie Recommendation System

Overview

The Movie Recommendation System is developed using PySpark and Flask. It provides personalized movie recommendations based on genre and year.

Strategies Used

  • Collaborative filtering techniques for recommendation.
  • Content-based filtering to analyze movie genres and release years.
  • Integration of PySpark for distributed processing of large datasets.
  • Implementation of Flask for building the web application.

Dataset

The dataset used for training the recommendation system includes movie ratings, titles, genres, and tags. It is sourced from the MovieLens dataset, which contains a large collection of movie ratings and metadata. Download Link
